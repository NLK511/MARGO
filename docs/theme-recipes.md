# Theme recipes

Theme recipes map visual personality to reusable runtime output.

## Key files
- `packages/themes/src/theme-family.ts`
- `packages/themes/src/theme-migration.ts`
- `packages/themes/src/theme-recipe.ts`
- `packages/themes/src/theme-runtime.ts`

## Notes
- Theme families are reusable presets.
- Tenants pick a family and override branding.
- Runtime output must stay deterministic.
