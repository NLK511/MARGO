# Design system

MARGO design tokens live in `packages/design`.

## Source of truth
- doctrine: `packages/design/src/doctrine.ts`
- tokens: `packages/design/src/tokens/*`
- recipes: `packages/design/src/recipes/*`
- validators: `packages/design/src/validators/*`

## Rules
- Use semantic tokens, not arbitrary tenant CSS.
- Keep typography role-based.
- Keep colors contrast-safe.
- Prefer curated layout controls.
