# Theme Studio operator guide

## What it does
Global Admin uses Theme Studio to create and publish reusable theme families.

## Safe workflow
1. Create a draft family or open a built-in theme.
2. Preview the theme matrix.
3. Edit tokens, spacing, typography, layout, and assets.
4. Run publish gates.
5. Save or publish.

## Operational rules
- Do not add tenant-specific branding to reusable themes.
- Keep changes token-based.
- Built-in themes are editable, but their changes are stored as local overrides and they cannot be deleted.
- Review audit logs for theme changes.
